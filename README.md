# LinearVision

线性代数几何变换可视化工具 — 通过交互式 2D/3D 图形直观理解矩阵变换、特征值与奇异值分解。

## 功能模块

### Module 1 · 基础空间变换 & 矩阵乘法合成

将矩阵作用于空间中的网格与基向量，实时观察线性变换的几何效果。

- **基础模式**：单独观察矩阵 A 的变换（B = I 时）
- **合成模式**：输入矩阵 A 和 B，观察 C = B × A 的连续过渡动画
  - 虚线箭头 = A 的基向量，实线箭头 = B×A 的基向量
  - 时间轴控制动画进度，支持 0.25x–2x 变速
- **2D 模式**：Canvas 渲染参考网格（虚线）与变换后网格（实线）
- **3D 模式**：Three.js 线框网格 + 基向量箭头，支持轨道旋转/缩放
- 支持鼠标滚轮缩放，自动适配变换范围

### Module 2 · 特征值与特征向量探测

交互式寻找矩阵 C = B × A 的特征向量，理解"不变方向"的几何含义。

- 显示特征值与特征向量（支持复数特征值）
- **向量拨盘**：拖拽输入向量 **v**，实时观察输出向量 **Cv** 的关系
- 自动吸附检测：当 **v** 与特征向量方向重合时高亮提示，显示对应特征值
- 复数特征值时提示"无实特征向量"

### Module 3 · 奇异值分解 (SVD) 动态解构

将矩阵 B 分解为 `B = U Σ Vᵀ`，展示从 A 到 B×A 的 SVD 阶段过程。

- **阶段 1** — Vᵀ × A（旋转/反射 A）
- **阶段 2** — Σ × Vᵀ × A（沿轴拉伸/压缩）
- **阶段 3** — U × Σ × Vᵀ × A = B × A（最终合成）
- 虚线箭头 = A 的基向量，实线箭头 = 当前有效矩阵的基向量
- 播放/暂停/拖拽时间轴，支持 0.25x–2x 变速
- 显示奇异值、秩，非满秩时提示维度坍缩
- B = I 时静态展示全量结果

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

开发服务器默认运行在 `http://localhost:3000`。

## 使用方式

1. 点击顶部 **2D / 3D** 切换维度
2. 在矩阵输入框 **A** 和 **B** 中编辑数值，或点击 **预设** 按钮加载示例变换
3. 结果矩阵 **B×A** 实时显示在控制栏
4. 三个模块面板实时响应矩阵变化：
   - **Module 1** — 观察网格和基向量的变形；B ≠ I 时播放 A → B×A 合成动画
   - **Module 2** — 拖拽蓝色向量寻找 B×A 的特征方向
   - **Module 3** — 播放 SVD 分解动画（B 的 SVD 阶段应用于 A）
5. 所有面板支持折叠，3D 视图支持鼠标旋转/缩放

## 技术栈

| 层面 | 技术 |
|------|------|
| 框架 | React 19 + TypeScript |
| 构建 | Vite 6 |
| 2D 渲染 | Canvas 2D API |
| 3D 渲染 | Three.js |
| 数学库 | 自行实现（~500 行纯 TS，零依赖） |

数学库涵盖：矩阵运算、特征值/特征向量（2×2 二次公式、3×3 三次方程 Cardano 法）、SVD 分解（2×2 闭式解、3×3 Jacobi 迭代），以及 Rodrigues 旋转公式用于正交矩阵插值。

## 项目结构

```
src/
├── math/          # 纯数学库：矩阵、特征值、SVD、预设
├── store/         # React Context 全局状态
├── hooks/         # useCanvas2D（2D 渲染循环）、useThreeScene（3D 场景）
├── components/    # 共享 UI：矩阵输入、预设按钮、维度切换、面板
├── modules/
│   ├── module1-space-transform/  # 模块1：空间变换
│   ├── module2-eigen/            # 模块2：特征值探测
│   └── module3-svd/              # 模块3：SVD 解构
└── utils/         # 缓动函数、几何工具
```
